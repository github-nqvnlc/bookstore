# Bookstore: An online bookstore website.


## Available Scripts

- In the `Client` project, you can run terminal:
  - ### `cd client`
  - ### `npm install`
  - ### `npm start`
 
Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.

- In the `Server` project, you can open new terminal:
  - ### Open Database use `XamPP` to run `MySQL`
  - ### `cd server`
  - ### `npm install`
  - ### Initialize the database `npx sequelize-cli db:migrate`
  - ### `npm start`
